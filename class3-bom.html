BOM - Browser object model

The Browser Object Model allows js to talk to the browser

The Window Object

The Window Object supported by all browser, It represents the browser's window.

All Global objects,functions and variables are property and element of Window Object,event the "DOM" is also a property of Window object.

Window Methods:-

Their are uncountable number of window methods that let us to talk with browser.some usefull Window methods are listed with short description bellow.

  • window.open() - open a new window/browser tab.
  • window.close() - close the window/browser tab.
  • window.moveTo() - move the current window.
  • window.resizeTo() - resize the current window.

Window Screen

window.screen object contains info about the users's screen.it also can be written without "window" prefix.

  • screen.width - get the width of current window
  • screen.height - get the height of current window
  • screen.availWidth - get the width of cureent window screen without interface like windows taskbar etc.
  • screen.availHeight - get the height of cureent window screen without interface like windows taskbar etc.

Window Location

window.location object can be used to get the current page location info and redirect users to any other page.same like window.screen , window.location also can be written without "window" prefix.

  • window.location.href - get the url of current window.
  • window.location.hostname - get the domain name of current window.
  • window.location.pathname - get the path and filename of current window.
  • window.location.protocol - get the web protocol used (http: or https:).
  • window.location.assign() - loads a new document.
  • window.location.port - get internet host port of current page.

Window History

window.history object contains the browser history and it is also can be written without "window" prefix. To protect the privacy of the users, there are limitations to how JavaScript can access this object.

  • window.history.back() - same like clicking back in the browser.
  • window.history.forward() - same like clicking forward in the browser.

Window Navigator

window.navigator contains info about the visitor's browser.it is also can be written without "window" prefix.

  • window.navigator.cookieEnabled - return is cookies is enabled or not with boolean value.
  • window.navigator.appCodeName - return application code name (it is now removed in leatest web standart.)
  • window.navigator.appVersion - return the browser verson information.
  • window.navigator.userAgent - return the user agent header sent by the browser to the server.
  • window.navigator.platform - return the operating system.
  • window.navigator.language - return the browser language.
  • window.navigator.onLine - return is the browser online or not.
  • window.navigator.javaEnabled() - return is the java is enabled or not.
